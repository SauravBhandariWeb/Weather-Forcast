# Weather-Forcast

- 🌍 Search weather by city name  
- 🌡️ Displays temperature (current, min, max)  
- 💧 Shows humidity levels  
- 🌬️ Displays wind speed  
- 📊 Shows atmospheric pressure  
- ⚡ Real-time data using API  
- 🎨 Clean and responsive UI  

---

## 🛠️ Technologies Used

- **Python** – Backend logic  
- **Flask** – Web framework  
- **HTML** – Structure of the web page  
- **CSS** – Styling and design  
- **OpenWeatherMap API** – Weather data provider  

---

## 📂 Project Structure


project/
│── app.py
│── templates/
│ └── index.html
│── static/
│ └── style.css


---

## ⚙️ How It Works

1. User enters a city name  
2. Request is sent to Flask backend  
3. Backend calls OpenWeatherMap API  
4. API returns weather data (JSON format)  
5. Data is processed and sent to frontend  
6. Weather details are displayed on the webpage  

---

## ▶️ How to Run the Project ##

**Run the app in the terminal:

python3 app.py
